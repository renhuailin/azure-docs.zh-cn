# <a name="azure-maps-routing-coverage"></a>Azure Maps 路由覆盖区域

本文提供了有关 Azure Maps 路由的覆盖区域信息。 发生搜索查询时，Azure Maps 返回从位置 A 到位置 B 的最佳路由。提供准确的旅行时间、旅行信息的实时更新以及路线说明。 你还可以添加其他搜索参数，如当前的交通状况、车辆类型和需要避免的路况。 路由的优化取决于区域。 这是因为，对于不同区域，Azure Maps 所具有的信息和相关准确性有所不同。 下表列出了可为其请求的区域和信息类型。
 
查看 [**地理编码**](geocoding-coverage.md)的覆盖区域。       
查看 [**流量**](traffic-coverage.md)的覆盖区域。  
查看[呈现](render-coverage.md)的覆盖区域。
