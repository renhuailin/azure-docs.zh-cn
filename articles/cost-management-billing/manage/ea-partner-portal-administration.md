# <a name="azure-ea-portal-administration-for-partners"></a>面向合作伙伴的 Azure EA 门户管理

本文介绍合作伙伴 EA 管理员可在 Azure EA 门户 (https://ea.azure.com) 中完成的常见任务。

## <a name="manage-partner-administrators"></a>管理合作伙伴管理员

Azure EA 门户中的每个合作伙伴管理员都能够添加或删除其他合作伙伴管理员。 合作伙伴管理员与间接注册的合作伙伴组织关联，不直接与注册关联。

### <a name="add-a-partner-administrator"></a>添加合作伙伴管理员

要查看与当前用户相同的合作伙伴组织关联的所有注册的列表，请选择“注册”选项卡，然后选中所需的注册框。

1. 以合作伙伴管理员身份登录。
1. 在左侧导航区域中选择“管理”。
1. 选择“合作伙伴”选项卡。
1. 选择“+ 添加管理员”并填写电子邮件地址、通知联系人和通知详细信息。
1. 选择 **添加** 。

### <a name="remove-a-partner-administrator"></a>添加合作伙伴管理员

要查看与当前用户相同的合作伙伴组织关联的所有注册的列表，请选择“注册”选项卡，然后选中所需的注册框。

1. 以合作伙伴管理员身份登录。
1. 在左侧导航区域中选择“管理”。
1. 选择“合作伙伴”选项卡。
1. 在“管理员”部分下，选择与要删除的管理员相应的行。
1. 选择右侧的“X”符号。
1. 确认要删除。

## <a name="manage-partner-notifications"></a>管理合作伙伴通知

合作伙伴管理员可以管理其注册使用情况通知的接收频率。 他们每周会自动收到未开票余额的通知。 他们可将每项通知的频率更改为每月、每周、每日，或完全禁用通知。

如果用户未收到某项通知，请使用以下步骤验证该用户的通知设置是否正确。

1. 以合作伙伴管理员的身份登录到 Azure EA 门户。
2. 依次选择“管理”、“合作伙伴”选项卡。 
3. 在“管理员”部分下查看管理员列表。
4. 若要编辑通知首选项，请将鼠标悬停在相应的管理员上，然后选择铅笔符号。
5. 根据需要提高通知频率并发出生命周期通知。
6. 根据需要添加联系人，然后选择“添加”。
7. 选择“保存”。

## <a name="view-enrollments-for-partner-administrators"></a>合作伙伴管理员查看合约

合作伙伴管理员可以在 Azure EA 门户中查看其所有直接和间接合约的列表视图。 将显示包含每个合约的概述的框以及合约编号、合约名称、余额和超额金额。

### <a name="view-a-list-of-enrollments"></a>查看合约列表

1. 以合作伙伴管理员身份登录。
1. 在页面左侧的导航栏上选择“管理”。
1. 选择“注册”选项卡。
1. 选中注册对应的框。

含所有合约的视图保留在页面顶部，其中每个合约都有一个框。 此外，还可以通过选择页面左侧导航中的当前注册编号，在注册之间导航。 系统将显示一个弹出窗口，使你能够通过选择相应的框来搜索注册或选择其他注册。
