# <a name="categories-and-subcategories-in-the-commercial-marketplace"></a>商业市场中的类别和子类别

根据解决方案类型，类别和子类别将映射到一个或两个[商业市场在线商店](overview.md#commercial-marketplace-online-stores)。 当你在合作伙伴中心创建商业市场产品/服务时，它可能会根据产品/服务类型、类别和你选择的其他选项在任一在线商店或同时在两个在线商店列出。 在极少数情况下，产品/服务与两个在线商店的用户都相关，它便可以同时在两个在线商店中显示。

选择最符合解决方案的类别和子类别。 分配适合产品/服务的类别和子类别，有助于客户在商业市场中找到你的产品/服务。 

可以选择：

- 至少一个类别，最多两个。 可以选择主类别和次要类别。
- 每个主类别最多包含两个子类别。 如果未选择任何子类别，则你的产品/服务在所选类别下仍不可发现。
