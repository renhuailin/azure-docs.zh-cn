# <a name="azure-policy-pattern-parameters"></a>Azure Policy 模式：参数

可以将策略定义设置为动态，减少使用[参数](../concepts/definition-structure.md#parameters)所需的策略定义的数量。 参数在策略分配过程中定义。 参数有一组预定义的属性，这些属性描述参数及其使用方式。
