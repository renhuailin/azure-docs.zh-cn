# <a name="enroll-your-subscription-in-per-user-access-pricing"></a>按用户访问定价注册订阅

在外部用户可以连接到部署之前，你需要在每用户访问定价中注册订阅。 每用户访问定价允许组织外部的用户访问订阅中的应用和桌面。 注册的订阅将根据连接到 Azure 虚拟桌面资源的不同用户的数量按月收费。

## <a name="how-to-enroll"></a>如何注册

将 Azure 订阅注册为每用户访问定价：

1. 在 [https://portal.azure.com](https://portal.azure.com/) 中登录 Azure 门户。

2. 在搜索栏中输入“Azure 虚拟桌面”，然后在“服务”下找到并选择“Azure 虚拟桌面” 。

3. 在“Azure 虚拟桌面”概述页中，选择“每用户访问定价”。

4. 在订阅列表中，选择要在其中部署 Azure 虚拟桌面资源的订阅。

5. 选择“注册”。

6. 查看产品条款，然后选择“注册”开始注册。 注册过程可能需要一小时才能完成。

7. 注册完成后，检查“订阅”列表中“每用户访问定价”列中的值，确保它已从“正在注册”更改为“已注册”。
