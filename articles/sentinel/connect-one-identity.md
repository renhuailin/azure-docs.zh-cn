# <a name="connect-one-identity-safeguard"></a>连接 One Identity Safeguard
One Identity Safeguard 数据连接器通过特权会话保护措施特定的仪表板增强了标准通用事件格式 (CEF) 连接器。 使用此连接器可以轻松开始使用设备生成的事件进行可视化、警报、调查等。

[!INCLUDE [reference-to-feature-availability](includes/reference-to-feature-availability.md)]

## <a name="prerequisites"></a>先决条件

- Azure Sentinel 工作区的读取和写入权限。

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>将 Azure Sentinel 连接到 One Identity Safeguard

1. 在 Azure 门户中，导航到“Azure Sentinel” > “数据连接器”，然后选择“One Identity Safeguard”连接器。

2. 选择“打开连接器页”。

3. 按照“One Identity Safeguard”页上的说明进行操作。
