在“新建表”对话框中填充表单，如下所示。

1. 输入名称“WeatherData”作为“表 ID”。  这是表的名称。
1. 对于此示例，请在“表吞吐量(自动缩放)”下选择“手动”。
1. 在估计的 RU/秒下，使用默认值 400。
1. 选择“确定”按钮以创建表。
