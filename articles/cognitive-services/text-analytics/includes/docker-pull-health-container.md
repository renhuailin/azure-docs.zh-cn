使用 [`docker pull`](https://docs.docker.com/engine/reference/commandline/pull/) 命令从 Microsoft 公共容器注册表下载此容器映像。

```
docker pull mcr.microsoft.com/azure-cognitive-services/textanalytics/healthcare:latest
```
