# <a name="language-support-for-computer-vision"></a>计算机视觉的语言支持

计算机视觉的某些功能支持多种语言；此处未提及的任何功能均只支持英语。

## <a name="optical-character-recognition-ocr"></a>光学字符识别 (OCR)

计算机视觉的 OCR API 支持多种语言。 Read 可以从使用混合语言的图像和文档中提取文本，无需语言参数即可包括相同的文本行。 有关详细信息，请参阅[光学字符识别 (OCR) 概述](overview-ocr.md)。

> [!NOTE]
> **可选语言参数**
>
> Read API 使用通用脚本模型来提取图像或文档中的所有多语言文本，包括混合语言文本行。 不要提供语言参数，除非你确定图像或文档中只有一种语言。 否则，服务可能会返回不完整和不正确的文本。

有关使用预览版语言和功能的说明，请参阅[如何指定模型版本](./Vision-API-How-to-Topics/call-read-api.md#determine-how-to-process-the-data-optional)。 预览版模型包含当前 GA 版本的任何增强功能。
