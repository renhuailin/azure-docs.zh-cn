托管磁盘的 ZRS 具有以下限制：

- 仅支持高级固态硬盘 (SSD) 和标准 SSD。
- 目前仅在美国西部 2、西欧、北欧和法国中部区域可用。
- 当前不能与 Azure 备份或 Azure Site Recovery 配合使用。
