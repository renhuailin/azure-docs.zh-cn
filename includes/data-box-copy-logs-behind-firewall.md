如果在 Data Box 的存储帐户上设置防火墙规则，你可能无法使用 Azure 门户“概述”窗格中的“复制日志路径”访问复制日志 。 若要访问日志，请修改存储防火墙设置以允许当前系统，或使用防火墙网络中的系统。
