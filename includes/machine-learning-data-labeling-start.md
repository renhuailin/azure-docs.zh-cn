标记项目是通过 Azure 机器学习管理的。 可以使用“数据标签”页来管理项目。

如果数据已在 Azure Blob 存储中，在创建标记项目之前，应以数据存储的形式提供这些数据。 有关如何使用数据存储的示例，请参阅[教程：创建第一个图像分类标记项目](../articles/machine-learning/tutorial-labeling.md)。
