若要购买设备，请转到 Azure Edge Hardware Center（预览版）并下订单。 Azure Edge Hardware Center 服务使你可以根据业务需求在各种 Azure Stack Edge SKU 中进行选择。 可以订购某种设备类型的多个单元、将多个设备运送到不同的位置、保存地址供将来订购，还可以跟踪订单的状态。 

交付订单后，可以配置设备并创建 Azure Stack Edge 资源来管理设备。
